Shaft Modal Analysis
~~~~~~~~~~~~~~~~~~~~

Visualize a full cyclic model

.. code-block:: default

    import pyansys







load a sector modal analysis file


.. code-block:: default

    rotor = pyansys.download_sector_modal()
    print(rotor)






.. rst-class:: sphx-glr-script-out

 Out:

 .. code-block:: none

    Units       : User Defined
    Version     : 15.0
    Cyclic      : True
    Result Sets : 48
    Nodes       : 460
    Elements    : 210


plot the rotor


.. code-block:: default

    rotor.plot(smooth_shading=True)





.. image:: /examples/00-read_binary/images/sphx_glr_sector_model_001.png
    :class: sphx-glr-single-img




plot a sector of the rotor


.. code-block:: default

    rotor.mas_grid.plot(color='w', smooth_shading=True)
